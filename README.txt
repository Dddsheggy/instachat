编写和调试环境：
*Visual Studio2019

运行注意事项：
*请注意测试的两台电脑都需要关闭防火墙！！！

文件说明：
*计网大作业 说明文档.pdf为程序的使用说明
*instachat\instachat.sln为解决方案文件
*instachat\instachat为工程文件夹
*instachat\instachat\Debug\instachat.exe为可执行程序
*instachat\instachat\Debug\*.txt为已保存的用户好友记录，
新生成的用户好友记录也会保存在这里
*instachat\instachat\Debug\trash文件夹保存拒收的文件
